HTML + CSS Cheat Sheet

Table HTML

Name Description Tag
Table Define a table <table>
Table Row Define a table row <tr>
Table Data Define a table cell <td>
Table Header Data Define a table header cell <th>
Table Header Define a group of a table header <thead>
Table Body Define a group of a table body <tbody>
Table Footer Define a group of a table footer <tfooter>

Table CSS

Name Description Format Example
Border Sets a border on the table border: 'size' 'type' 'color' border: 1px solid white
Border Collapse Removes the borders from within each cell border-collapse: 'state' border-collapse: collapse
Width Defines the width of a table. width: % or px width: 100%
